GetFeatureByPost

  • GetFeatureByPost(option?): Promise<LayerPropertySheetInfo>

  • Post方式查询矢量图层Features（cql暂不支持TOUCHES|CROSSES|OVERLAPS|DWITHIN|BEYOND方法）

    参数

    • 可选option: {
          cql?: string;
          maxFeatures?: number;
          propertyname?: string;
          startIndex?: number;
          typename?: string;
          workspace?: string;
          workspaceUri?: string;
      }
      • 可选cql?: string
      • 可选maxFeatures?: number
      • 可选propertyname?: string
      • 可选startIndex?: number
      • 可选typename?: string
      • 可选workspace?: string
      • 可选workspaceUri?: string

    返回 Promise<LayerPropertySheetInfo>

    示例

    import wfsHelper from 'geoserver-helper/wfs'
    const wfsHelperInstance = new wfsHelper({
      url: "/geoserver/wfs",
      workspace: "qhd",//这个工作空间优先级最低，如果参数中没传的话就会用它
    });
    wfsHelperInstance.GetFeatureByPost({
      "workspace": "qhd",//工作空间（如果是默认的工作空间可以不传）
      "workspaceUri": "http://qhd",//工作空间uri（如果是默认的工作空间可以不传）
      "typename": "qhd:永久基本农田", 
      "startIndex": 0, 
      "maxFeatures": 10, 
      "cql": "type = '种植非粮食作' AND INTERSECTS (the_geom, MULTIPOLYGON(((119.149559 40.60191,119.1549 40.597565,...119.176018 40.609817))))"
    }).then(res => {
      console.log(res)
    })
